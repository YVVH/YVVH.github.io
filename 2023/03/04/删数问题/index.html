删数问题

题目描述

键盘输入一个高精度的正整数 \(N\)（不超过 \(250\) 位），去掉其中任意 \(k\) 个数字后剩下的数字按原左右次序将组成一个新的非负整数。编程对给定的 \(N\) 和 \(k\)，寻找一种方案使得剩下的数字组成的新数最小。

输入格式

输入两行正整数。

第一行输入一个高精度的正整数 \(n\)。

第二行输入一个正整数 \(k\)，表示需要删除的数字个数。

输出格式

输出一个整数，最后剩下的最小数。

样例 #1

样例输入 #1

样例输出 #1

Solution

为了使剩下的数尽可能的小，只需要考虑高位的改变情况，

例如\(n=175438\)时，删数顺序为\(7,5,4,8\)，解为\(13\)，

因此每次从高位遍历，找出第一次出现降序的位置删掉大的数即可。
